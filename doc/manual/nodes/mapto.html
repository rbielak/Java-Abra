Map-to

Description

Define a Table or Xml node for a class to map to. Found inside class nodes.

Parameters

Attribute Description Required
table the table name for objects of this type. One of these.
node xml-node name (defaults to class name)

Parameters specified as nested elements

NONE

Examples

  <map-to table="foo_user" />
