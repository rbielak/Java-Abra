Class-Implements

Description

Defines interfaces for every class to implements

Parameters

Attribute Description Required
implements the name of the classes to implement Yes

Parameters specified as nested elements

NONE

Examples

  <class-implements implements="java.io.Serializable, org.foo.Bar" >

ALL classes in this map-file will implement the named classes
